Step 1: Install Dependencies
├─ cd /Users/macbookair/Documents/GitHub/SafeTech
└─ npm install

Step 2: Configure Environment
├─ cp .env.example .env
└─ Edit .env with your settings (database URL, API keys, etc.)

Step 3: Start MongoDB
├─ mongod
└─ (Keep this running in a separate terminal)

Step 4: Run Application
├─ npm run dev          (Development mode with auto-reload)
└─ OR npm start        (Production mode)

Step 5: Access Platform
├─ Open http://localhost:3000 in your browser
├─ Register a new account
├─ Or login with admin credentials
└─ Start adding laptops and plans!

🚨 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem: MongoDB connection error
├─ Solution 1: Start MongoDB with `mongod`
├─ Solution 2: Check MONGODB_URI in .env
└─ Solution 3: Use MongoDB Atlas (cloud)

Problem: Port 3000 already in use
├─ Solution 1: Kill process: lsof -ti:3000 | xargs kill -9
└─ Solution 2: Use different port: PORT=3001 npm run dev

Problem: Cannot find module 'express'
├─ Solution 1: Run `npm install`
├─ Solution 2: Delete node_modules and package-lock.json, reinstall
└─ Solution 3: Check Node.js version

Problem: Email not sending
├─ Solution 1: Verify Gmail app password
├─ Solution 2: Check EMAIL_USER and EMAIL_PASSWORD in .env
└─ Solution 3: Enable 2-Step Verification on Gmail
