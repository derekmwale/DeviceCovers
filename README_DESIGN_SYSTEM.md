## 💡 Customization Examples

### Change Primary Color
```css
/* In premium.css */
--primary: #001a4d;        /* Navy */
--primary-dark: #000d2e;
--primary-light: #e6ebf5;

/* Change to your color, e.g., Purple */
--primary: #6366f1;        /* Indigo */
--primary-dark: #4f46e5;
--primary-light: #e0e7ff;
```

### Change Accent Color
```css
/* Currently Gold */
--accent: #d4af37;

/* Change to Silver, etc */
--accent: #c0c0c0;
```

### Adjust Spacing
```css
/* Current base: 1rem */
--space-4: 1rem;

/* For more spacious design */
--space-4: 1.25rem;
```

### Modify Animation Speed
```css
/* Current: 250ms */
--transition-base: 250ms ease-in-out;

/* Make faster */
--transition-base: 150ms ease-in-out;
```
